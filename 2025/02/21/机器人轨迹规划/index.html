梯形速度曲线轨迹

梯形速度曲线轨迹规划是工业控制领域内应用最为广泛的速度控制策略之一，标准形式的梯形速度规划包含匀加速，匀速和匀减速三个阶段，在变速过程中，加速度始终是人为设定的固定值。

  • 匀加速：加速度是恒定的正值，速度随着时间线性增加
  • 匀速：加速度为 0，速度大小恒定
  • 匀减速：加速度是恒定的负值，速度随着时间线性减小

需要定义一些曲线轨迹的参数

  • 初始位置 $q_0$ ，速度 $v_0$
  • 结束位置 $q_1$ ，速度 $v_1$
  • 最大加速度 $a_{a}$ ，最大减速度 $a_d$
  • 运行过程中期望最大速度 $v_{f}$
  • 运行过程中实际最大速度 $v_{max}$
  • 匀加速段时间 $T_a$ ，匀减速段时间 $T_d$

轨迹设计

  1. 首先根据需要运行的距离来计算运行过程中是否具有匀速段，需要计算当只有匀加速和匀减速段时，达到最大速度时运行的距离 $\frac{v_{f}^2-v_0^2}{2a_a}+\frac{v_{1}^2-v_{f}^2}{2a_d}$ ，如果这个距离不大于期望距离 $\vert q_1-q_0\vert$ ，那就没有匀速段，否则就存在匀速段
  2. 对于有匀速段的，计算各个段的位移和时间，注意此时实际最大速度达到了期望最大速度
    • 加速段： $T_a=\frac{v_{f}-v_0}{a_a},x_a=\frac{v_{f}^2-v_0^2}{2a_a}$
    • 匀速段： $T_v=\frac{\vert q_1-q_0\vert-(\frac{v_{f}^2-v_0^2}{2a_a}+\frac{v_{1}^2-v_{f}^2}{2a_d})}{v_{f}},x_v=\vert q_1-q_0\vert-(\frac{v_{f}^2-v_0^2}{2a_a}+\frac{v_{1}^2-v_{f}^2}{2a_d})$
    • 减速段： $T_d=\frac{v_1-v_{f}}{a_d},x_d=\frac{v_{1}^2-v_{f}^2}{2a_d}$
  3. 对于没有匀速段的，各个段的位移和时间，此时最大速度并未达到期望最大速度 $v_{max}=\frac{\sqrt{-(a_a-a_d)(a_dv_0^2-a_av_1^2+2a_aa_d\vert q_1-q_0\vert)}}{a_a-a_d}$
    • 加速段： $T_a=\frac{v_{max}-v_0}{a_a},x_a=v_0T_a+\frac{1}{2}a_aT_a^2$
    • 减速段： $T_d=\frac{v_1-v_{max}}{a_d},x_d=\vert q_1-q_0\vert-x_a$

  • 无匀速段

优点

  • 计算简单，易于实现
  • 运动过程平滑，适合大多数工业应用
  • 可以通过调整加速度和最大速度来优化运动性能

缺点

  • 加速度突变可能导致机械振动
  • 对于高动态要求的场景，可能需要更复杂的轨迹规划方法（如S形速度曲线）

双 S 形速度曲线轨迹

双S形速度曲线轨迹，也被称为七段式轨迹或铃型规划（Bell），是一种由七段加加速度恒定的轨迹组成的平滑速度曲线。与梯形速度曲线相比，双S形速度曲线轨迹更加平滑，因为它将梯形轨迹的加速和减速段进一步细分为三段，从而避免了加速度的不连续变化

双 S 形速度曲线轨迹分为七个部分

  1. 加速段初期：加速度从 0 线性增加到最大加速度 $a_a$ ，加加速度保持恒定 $J$ ，时间为 $T_1$
  2. 加速段中期：加速度保持在最大加速度 $a_a$ ，时间为 $T_2$
  3. 加速段末期：加速度从最大加速度线性减小到 0，加加速度保持恒定 $-J$ ，时间为 $T_3$
  4. 匀速段：速度保持不变 $v_m$ ，加速度保持为 0，加加速度保持为 0，时间为 $T_4$
  5. 减速段初期：加速度从 0 线性减小到最大减速度 $a_d$ ，加加速度保持恒定 $-J$ ，时间为 $T_5$
  6. 减速段中期：加速度保持在最大减速度 $a_d$ ，时间为 $T_6$
  7. 减速段末期：加速度从最大减速度线性增加到 0，加加速度保持恒定 $J$ ，时间为 $T_7$

曲线轨迹设计

  1. 判断是否有加速段中期：当只有加速段初期和末期时计算得到的最大速度 $v=\frac{a_a^2}{J}$
    • 如果 $v<v_m$ 则有加速段中期
      • 加速段初期时间 $T_1=\frac{a_a}{J}$ ，距离为 $x_1=\frac{a_a^3}{6J^2}$
      • 加速段中期时间 $T_2=\frac{v_m-v}{a_a}$ ，距离为 $x_2=\frac{JT_1^2T_2+a_aT_2^2}{2}$
      • 加速段末期时间 $T_3=\frac{a_a}{J}$ ，距离为 $x_3=(\frac{JT_1^2}{2}+a_aT_2)T_3+\frac{a_aT_3^2}{2}-\frac{JT_3^3}{6}$
      • 则整个加速段的路程为 $x_a=x_1+x_2+x_3$
    • 如果 $v\geq v_m$ 则无加速段中期，在中间达到最大加速度为 $a_m=\sqrt{Jv_m}$
      • 加速段初期时间 $T_1=\frac{a_m}{J}$ ，距离为 $x_1=\frac{a_m^3}{6J^2}$
      • 加速段末期时间 $T_3=\frac{a_m}{J}$ ，距离为 $x_3=\frac{JT_1^2T_3}{2}+\frac{a_aT_3^2}{2}-\frac{JT_3^3}{6}$
      • 则整个加速段的路程为 $x_a=x_1+x_3$
  2. 判断是否有减速段中期，与加速段一致，计算减速的最大速度为 $v=\frac{a_d^2}{J}$
    • 如果 $v<v_m$ 则有减速段中期
      • 减速段初期时间 $T_5=\frac{a_d}{J}$ ，距离为 $x_5=v_mT_5-\frac{JT_5^3}{6}$
      • 减速段中期时间 $T_6=\frac{v_m-v}{a_d}$ ，距离为 $x_6=(v_m-\frac{JT_5^2}{2})T_6+\frac{a_dT_6^2}{2}$
      • 减速段末期时间 $T_7=\frac{a_d}{J}$ ，距离为 $x_7=(v_m-\frac{JT_5^2}{2}+a_dT_6)T_7+\frac{a_dT_7^2}{2}+\frac{JT_7^3}{6}$
      • 则整个减速段的路程为 $x_d=x_5+x_6+x_7$
    • 如果 $v\geq v_m$ 则无加速段中期，在中间达到最大加速度为 $a_m=-\sqrt{Jv_m}$
      • 减速段初期时间 $T_5=-\frac{a_m}{J}$ ，距离为 $x_5=v_mT_5-\frac{JT_5^3}{6}$
      • 减速段末期时间 $T_7=-\frac{a_m}{J}$ ，距离为 $x_7=(v_m-\frac{JT_5^2}{2})T_7+\frac{a_mT_7^2}{2}+\frac{JT_7^3}{6}$
      • 则整个减速段的路程为 $x_d=x_5+x_7$
  3. 判断是否有匀速段：计算加速段和减速段走过的距离 $x=x_a+x_d$
    • 如果期望距离 $x_f\leq x$ ，如果有加减速段中期的话，去掉加减速段中期，重新计算路程 $x$ ，再次判断。如果 $x_f\leq x$ 那么就没有匀速段
      • 根据上述可知 $x_f=x_a+x_d$ ，此时加速段距离为 $x_a=\frac{5a_m^3}{6J^2}$ ，减速段距离为 $x_d=\frac{2a_m^3}{J^2}$ （注意此处的 $a_m=\sqrt{Jv_m}$ ）
      • 计算出最大的加速度和减速度为 $a_m= \sqrt[3]{\frac{3J^2x_f}{4}}$ ，带入之后就可以计算出各个阶段的时间
    • 如果期望距离 $x_f>x$ 那么就有匀速段
      • 匀速段距离为 $x_m=x_f-x$
      • 匀速段时间为 $T_4=\frac{x_m}{v_m}$
