超前和滞后补偿器

超前补偿器和滞后补偿器是控制系统中常用的两种动态补偿器，通过调整系统的频率响应来优化稳定性、响应速度和稳态误差等指标

根轨迹与系统性能

  • 稳定性
    • 若所有极点位于复平面的左半部分，系统稳定
    • 若有极点位于右半部分，系统不稳定
  • 动态响应
    • 超调量：极点距离虚轴越近，阻尼越小，超调量越大
    • 调节时间：极点离虚轴越远，调节时间越短
    • 震荡频率：极点的虚部越大，震荡频率越高
  • 如果特征方程的根为复数 s=σ±jws=\sigma\plusmn jws=σ±jw
    • 复数极点的实部 σ\sigmaσ 决定了系统的收敛速度和调节时间
    • 复数极点的虚部 www 决定了系统的震荡频率
  • 主导极点：靠近虚轴的极点对系统响应起主导作用，远离虚轴的极点影响较小，若存在一对共轭复数主导极点，系统的动态响应主要由这对极点决定
  • 通过调整增益 KKK 可以使得系统具有较小的超调量，较短的调节时间和较高的稳定性
  • 随着 KKK 不断增大，根从极点移动到零点的位置，就是闭环传递函数极点的位置，然后经过判断闭环传递函数的极点位置来判断系统的稳定性
  • 对于在根轨迹上的点，满足 ∠KG(s)H(s)=−π(2q+1)q=0,±1,...\angle KG(s)H(s)=-\pi(2q+1)\quad q=0,\plusmn1,...∠KG(s)H(s)=−π(2q+1)q=0,±1,... ，也就是该点到零点的夹角和减去该点到极点的夹角和
  • 对于在根轨迹上的点，满足 ∣KG(s)H(s)∣=1=K∏lz∏lp\vert KG(s)H(s)\vert=1=K\frac{\prod l_z}{\prod l_p}∣KG(s)H(s)∣=1=K∏lp​∏lz​​ ，后面那一项就是该点到所有零点的距离之积除以该点到所有极点的距离之积

奈奎斯特稳定判据

柯西辐角原理

假设 f(x)f(x)f(x) 是一个闭合路径 C 上解析且无零点的复变函数，且在 C 内除有限个极点和零点外解析，则柯西辐角原理可以表述为

12π△Carg⁡f(s)=Z−P\frac{1}{2\pi}\triangle_C\arg f(s)=Z-P 2π1​△C​argf(s)=Z−P

  • △Carg⁡f(s)\triangle_C\arg f(s)△C​argf(s) 是函数 f(s)f(s)f(s) 的辐角 f(s)f(s)f(s) 的辐角沿闭合路径 C 的变化量
  • ZZZ 是函数 f(s)f(s)f(s) 在 C 内的零点个数
  • PPP 是函数 f(s)f(s)f(s) 在 C 内的极点个数

在 S 平面内顺时针画一条闭合曲线 A，而 B 曲线是 A 通过 f(s)f(s)f(s) 后再 f(s)f(s)f(s) 平面上的映射，A 曲线每包含一个 f(s)f(s)f(s) 的零点，B 曲线就绕着 (0,0)(0,0)(0,0) 顺时针绕一圈，A 曲线每包含一个 f(s)f(s)f(s) 的极点，B 曲线就绕着 (0,0)(0,0)(0,0) 逆时针绕一圈

奈奎斯特图

奈奎斯特稳定判据（Nyquist Stability Criterion）是控制理论中用于分析闭环系统稳定性的一种重要方法，通过分析开环传递函数 KG(s)H(s)KG(s)H(s)KG(s)H(s) 的频率响应即奈奎斯特图，判断闭环系统传递函数 KG(s)1+KG(s)H(s)\frac{KG(s)}{1+KG(s)H(s)}1+KG(s)H(s)KG(s)​ 的稳定性

对上述方程进行分析可知， KG(s)H(s)KG(s)H(s)KG(s)H(s) 的极点就是 1+KG(s)H(s)1+KG(s)H(s)1+KG(s)H(s) 的极点， 1+KG(s)H(s)1+KG(s)H(s)1+KG(s)H(s) 的零点就是闭环传递函数 KG(s)1+KG(s)H(s)\frac{KG(s)}{1+KG(s)H(s)}1+KG(s)H(s)KG(s)​ 的极点

柯西辐角原理被用于推导奈奎斯特稳定判据，具体步骤如下

  1. 对于闭环系统的特征方程 1+G(s)H(s)=01+G(s)H(s)=01+G(s)H(s)=0 ，定义函数 f(s)=1+G(s)H(s)f(s)=1+G(s)H(s)f(s)=1+G(s)H(s)
  2. 选择闭环路径，选择奈奎斯特路径 C，通常是一个包含整个右半平面的半圆，并分析 G(s)H(s)G(s)H(s)G(s)H(s) 在 C 上的行为
  3. 根据柯西辐角原理， ZZZ 是 f(s)f(s)f(s) 在右半平面的零点数，即闭环传递函数的不稳定极点数， PPP 是 f(s)f(s)f(s) 在右半平面的极点数，即开环传递函数的不稳定极点数
  4. 通过分析 f(s)=1+G(s)H(s)f(s)=1+G(s)H(s)f(s)=1+G(s)H(s) 的辐角变化，可以推导出奈奎斯特稳定判据 Z=P−NZ=P-NZ=P−N ，其中 NNN 就是曲线围绕 (0,0)(0,0)(0,0) 的包围次数，顺时针绕一圈就减一，逆时针绕一圈就加一
  5. 将上述 f(s)f(s)f(s) 函数左右减一，也就是将曲线平移一下，而原来的 ZZZ 和 PPP 保持不变，得到的这个曲线就是奈奎斯特图

奈奎斯特稳定性判据

由于系统稳定的条件就是闭环传递函数 G(s)1+G(s)H(s)\frac{G(s)}{1+G(s)H(s)}1+G(s)H(s)G(s)​ 在复平面的右半边没有极点，也就是 Z=0Z=0Z=0 ，根据上述得到的 Z=P−NZ=P-NZ=P−N 公式，可以得到奈奎斯特稳定性判据为 P=NP=NP=N

伯德图

伯德图定义

伯德图是系统频率相应的一种图示方法，伯德图由幅值图和相角图组成，两者都按照频数的对数分度绘制。幅值图的纵轴表示为 20log⁡A20\log A20logA 的形式，使用分贝来表示幅值

  • 幅频特性图
    • 横轴为频率，通常以对数刻度表示
    • 纵轴为系统幅值，以分贝形式表示
    • 展示系统增益随频率的变化
  • 相频特性图
    • 横轴为频率，通常以对数刻度表示
    • 纵轴为系统相位
    • 展示系统相位随频率的变化

超前补偿器定义

超前补偿器主要用于提高系统的相位裕度，增强系统的稳定性和响应速度，传递函数形式为

超前补偿器特点

  1. 增加相位裕度，提高系统的稳定性

滞后补偿器

滞后补偿器定义

滞后补偿器主要用于改善系统的稳态误差，同时保持系统的稳定性。其传递函数形式为

G(s)=s+zs+p∣z∣>∣p∣G(s)=\frac{s+z}{s+p}\quad\vert z\vert>\vert p\vert G(s)=s+ps+z​∣z∣>∣p∣

滞后补偿器特点

  • 改善稳态误差
  • 降低高频增益，减少噪声影响
  • 适用于系统稳态误差较大时

滞后补偿器示例

对于如下的一个系统

可以求解该系统的传递函数为 KG(s)1+KG(s)\frac{KG(s)}{1+KG(s)}1+KG(s)KG(s)​ 而系统的误差可以表示为 E=R−XE=R-XE=R−X ，带入传递函数可以得到

E=11+KG(s)RE=\frac{1}{1+KG(s)}R E=1+KG(s)1​R

当系统运行到稳定时，可以计算出系统的稳态误差如下，当 K=6K=6K=6 时，稳态误差为 0.5R0.5R0.5R

E=6K+6RE=\frac{6}{K+6}R E=K+66​R

所以可以设计一个滞后补偿器将稳态误差减小，设计补偿器如下

P(s)=s+zs+pP(s)=\frac{s+z}{s+p} P(s)=s+ps+z​

重新计算稳态误差为

E=11+KG(s)P(s)R=66+KzpRE=\frac{1}{1+KG(s)P(s)}R\\=\frac{6}{6+K\frac{z}{p}}R E=1+KG(s)P(s)1​R=6+Kpz​6​R

要使稳态误差减小，必须增大 KzpK\frac{z}{p}Kpz​ 项，所以这里设计的滞后补偿器需要 ∣z∣>∣p∣\vert z\vert>\vert p\vert∣z∣>∣p∣

当 p=0p=0p=0 时， zp→∞\frac{z}{p}\rightarrow\inftypz​→∞ 稳态误差就趋近于 0，这时候将滞后补偿器化简可以得到 P(s)=1+zsP(s)=1+\frac{z}{s}P(s)=1+sz​ ，第一项就是比例项，第二项是积分项，这就是个 PI 控制器，PI 控制器可以将系统的稳定误差控制趋近于 0 的

现在设计补偿器使得误差 E=0.1RE=0.1RE=0.1R ，并且保持 K=6K=6K=6 ，计算之后可以得到 zp=9\frac{z}{p}=9pz​=9 ，但是只有一个比例值，具体取值还需要进行判断和调整。可以选取 z=9,p=1z=9,p=1z=9,p=1 和 z=0.9,p=0.1z=0.9,p=0.1z=0.9,p=0.1 进行比较，得到如下的运行结果

其中蓝色的线就是 z=9,p=1z=9,p=1z=9,p=1 的情况，而红色的线为 z=0.9,p=0.1z=0.9,p=0.1z=0.9,p=0.1 的情况，可以看出红线与黄线的运行轨迹比较相似，而蓝线却很不一样，这里可以从根轨迹来分析这两种情况，如下，第一个是原系统的根轨迹，第二个是 z=9,p=1z=9,p=1z=9,p=1 的根轨迹，第三个是 z=0.9,p=0.1z=0.9,p=0.1z=0.9,p=0.1 的根轨迹

可以发现第二个根轨迹已经与原来的根轨迹相差很多了，这是因为，所有根轨迹上的点满足该点到零点的夹角和减去该点到极点的夹角和为 -180，所以当新增加的零极点相差很小时，它们两个点与根轨迹上点的夹角就很相近，计算的时候就会抵消，从而对原来的根轨迹的影响不大，反之就是导致根轨迹变化很大，从而丢失了系统原来的特性。所以在设计滞后补偿器时，要使得零极点尽量靠近虚轴，这样就算它们差别很大，也不会距离很远，就能尽可能保证系统原有的瞬态响应
